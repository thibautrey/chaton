# Getting Started

Chatons is a desktop AI workspace built with Electron, React, TypeScript, SQLite, and Pi Coding Agent.

## Who this documentation is for

- End users who want to use Chatons daily
- Contributors who want to understand the architecture
- Extension authors integrating custom capabilities

## Start here

- Read the [User Guide](/docs/user-guide) for product behavior
- Read the [Developer Guide](/docs/developer-guide) for implementation details
- Read the [Pi Integration](/docs/pi-integration) page for runtime specifics
