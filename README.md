# Dashboard Native

Electron + React + TypeScript + shadcn UI skeleton for macOS.

## Scripts

- `npm run dev`: run Vite + Electron in development.
- `npm run build`: build renderer and Electron main/preload code.
- `npm run dist`: create macOS `.dmg` package in `release/`.

## Notes

- Current implementation is a static UI skeleton inspired by the provided screenshot.
- No app logic/API integration is included yet.
